alquipelis


Hola bienvenido a alquipelis en esta pagina de indicaciones te enseñare como registrar un producto pero primero te daré una descripción de mi empresa, en esta empresa nosotros nos encargamos de registrar nuevos productos que tú quieras además de eso tenemos un catalogo extenso de más de 50 películas de diversos tipos como acción, romance infantiles de comedia entre otros además de eso puedes buscar y visualizar una película en nuestro catalogo o buscar una película que tu quieras en nuestro repertorio.

Pasos para registrar un producto


Para registrar un nuevo producto sigue estos pasos:


  1. Nombre: Para este campo, solo ingresa el nombre del producto. Asegúrate de que el nombre no sea muy largo y no supere los 20 caracteres o letras, y que no sea un texto completo.

  2. Categoría: Para este campo, selecciona la categoría del producto de entre las opciones proporcionadas.

  3. Imagen: Para este campo, selecciona la imagen de tu producto y elige el nombre de la imagen de entre al menos diez opciones disponibles.

  4. Código del producto: Para este campo, ingresa un código que debe ser único para el producto. Es clave que el código tenga al menos 8 caracteres, incluyendo al menos una minúscula, una mayúscula y dos números.

  5. Precio: Para este campo, ingresa el precio del producto. Solo se aceptan números y debe estar en pesos colombianos (COP).

  6. Año de estreno: En este campo se debe colocar el año cuando salió la película.

  7. Duración: En este campo se colocará la duración total de la película.

  8. Por último, haz clic en el botón "Registrar" para completar tu proceso de registro.

  9. Advertencia: Si te equivocas en el proceso de registro, serás redirigido a esta página de indicaciones. Debes revisar atentamente los campos marcados y, si hay algún error, corrige los errores antes de volver a intentarlo.

  10. Si deseas registrar otro producto, puedes limpiar los campos haciendo clic en el botón "Limpiar Campos".

  11. Por último, una vez que el producto sea validado, serás redirigido a la página principal de productos.
